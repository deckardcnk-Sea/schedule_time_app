## 一、核心界面结构（视频里反复出现）

1. **顶部第一行**：左侧 4 个图标 + 右侧搜索框 + “今天”按钮
   - 图标 1：日历账户/日历列表（打开日历账户切换侧边栏）
   - 图标 2：邮件/邀请（收件箱，待处理邀请）
   - 图标 3：列表（打开“我的列表”分类侧边栏）
   - 图标 4：加号（新建事件/提醒事项）
2. **顶部第二行**：日 / 周 / 月 / 年 分段切换
3. **左侧大标题**：2024年6月 / 2024年7月
4. **周视图**：7 天日期头 + 农历 + 全天区域 + 时间块 + 当前时间红线
5. **月视图**：整月网格 + 日期 + 事件/周计划彩色标记
6. **年视图**：12 个月小月历概览
7. **手机端**：单日预览列表模式（竖屏）

## 二、日历/事件功能

8. **新建事件/提醒事项**：弹窗选择“日程”或“提醒事项”
9. **编辑事件**：标题、全天开关、开始/结束时间、重复规则、日历分类、受邀人
10. **编辑提醒事项**：标题、日历分类、提醒时间（无/日程当天/1天前/2天前/1周前）
11. **时间块**：不同颜色代表不同分类/日历
12. **全天事项**：顶部全天区域显示
13. **当前时间线**：红色贯穿线
14. **拖拽/跨天**：今天做不完拖到第二天（视频提到，但画面未展示操作细节）
15. **重复设置**：周期任务、结束重复规则

## 三、提醒事项/列表功能

16. **我的列表**：年计划、月计划、周计划、开心清单、书单、最近删除、自定义列表
17. **列表视图**：左侧分类、右侧任务列表
18. **任务字段**：标题、完成打勾、旗标、到期日/截止日期、所属列表
19. **智能筛选**：今天 / 计划 / 全部 / 旗标 / 完成 / 分配给我
20. **新建任务**：直接输入标题，选择列表

## 四、时间记录与统计

21. **时间记录**：通过时间块在日历上记录时间花在哪
22. **按分类/颜色看时间分布**：周视图一眼看出时间投入
23. **统计/复盘**：周复盘、完成度、成就感（视频提到，未展示具体界面）

## 五、系统与体验
